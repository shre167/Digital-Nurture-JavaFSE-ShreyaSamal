# Digital-Nurture-JavaFSE-ShreyaSamal

This repository contains the Week 1 assignments completed for Java programming, DSA exercises, design pattern implementations, and PL/SQL practice.

## What has been done

### Java DSA exercises
- Implemented inventory management using a map-based structure
- Implemented search and sorting programs
- Implemented employee, task, and library management examples
- Implemented financial forecasting logic

### Java Design Pattern exercises
- Implemented Singleton, Factory Method, Builder, Adapter, Decorator, Proxy, Observer, Strategy, Command, MVC, and Dependency Injection examples

### PL/SQL exercises
- Added schema creation script
- Added scripts for control structures, error handling, stored procedures, functions, triggers, cursors, and packages

## Project structure
- DSA/ - Data Structures and Algorithms Java programs
- DesignPattern/ - Java design pattern implementations
- PLSQL/ - Oracle PL/SQL scripts

## How to use
- Compile Java programs with:
  - javac *.java
- Run a Java program with:
  - java ClassName
- Execute PL/SQL scripts in an Oracle database environment.
